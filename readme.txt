【操作方式】
请切换到英文输入法下开始游戏
方向键切换选择/移动，空格确定，X键返回，非开始界面关闭游戏请用右上角

【温馨提醒】
每天只能外出一次，反过来说每外出一次增加一天，巡视基地不增长天数，每增加一天（外出返回基地时）将自动存档一次
背包（4格）里物品不可删除，物品每次返回基地时会自动安置，如果超过培养上限（植物5动物3），今天的可培养物品将不会放入培养室，可以选择取消已有动植物的培养以留出空位，珍宝没有上限
存档上限为2个，如果两个存档都有时再次开始新游戏并经过一天以上，将自动覆盖天数较小的存档，天数不会显示在存档界面，可以进游戏查看后关闭再进
游戏目标是集满图鉴，已培养完的物品或拾取到的珍宝可以开启图鉴
